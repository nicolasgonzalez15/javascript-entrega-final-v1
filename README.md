# Javascript Responsive v1
Proyecto con sitio web de proyecto final para curso Javascript de Tecno 3F

## Tecnologías utilizadas
* HTML y CSS para maquetación y estilos del sitio
* JS para las funcionalidades de API / Validación dinámica de formulario de contacto

## Estructura sitio
* Barra de navegación, con opciones
* Sección Principal, con detalle de sitio
* Sección Temáticas, con tópicos del sitio
* Sección API - Utilizando API de Rick and Morty - Buscador de personajes
* Sección contacto - Formulario dinámico con JS
